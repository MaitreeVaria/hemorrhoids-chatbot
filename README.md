# Patient RAG Chatbot - Hemorrhoids & Constipation

A compassionate AI assistant that helps patients manage hemorrhoids and constipation at home, with persistent conversation memory and red flag detection.

## Features

- Patient-friendly responses using Claude
- Persistent conversation memory across sessions
- Automatic red flag detection for concerning symptoms
- RAG system using your medical documents
- Secure, local storage
